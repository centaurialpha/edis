¿Qué es EDIS?

EDIS (de Entorno de Desarrollo Integrado Simple) es un IDE libre, gratuito y multiplataforma para el lenguaje de programación C potenciado por Python. Presenta una interfáz simple y amigable con funcionalidades avanzadas que ayudan al desarrollo.

Algunas características

Awesome editor!

EDIS posee un completo editor de código con resaltado
de sintáxis, plegado de código, resaltado de braces
({}, [], (), <>) búsqueda y reemplazo de palabras,
minimapa, auto-indentación, y más!

Static error checker

El analizador stático de errores detecta
principalmente los tipos de errores que el compilador
normalmente no los detecta, por ejemplo: índices fuera
de rango, advierte si se utilizan funciones obsoletas,
variables sin inicializar, funciones y variables sin
utilizar, pérdidas de memoria, etc.

Tree symbols

El árbol de símbolos es de gran ayuda, puedes moverte
hacia la definición con solo clicker sobre uno de
ellos. Soporta variables globales, funciones, uniones,
estructuras y miembros de uniones y estructuras.
