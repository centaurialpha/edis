## Dependencias

- [Python](http://python.org) >= 2.7
- [PyQt](http://www.riverbankcomputing.com/software/pyqt/intro) >= 4.8

## Ejecutar EDIS-C

Debes clonar el repositorio y ejecutarlo:

        git clone git://github.com/centaurialpha/edis.git
        cd edis
        python edis-c.py
